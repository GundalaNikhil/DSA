# STR-005: Sentence Reordering

## Problem Statement

Given a sentence, reverse the order of its words while keeping each word's spelling intact. Words are separated by one or more spaces. Output the reordered sentence with single spaces between words.

## Input Format

- Single line: string `s`

## Output Format

- The reordered sentence

## Constraints

- `1 <= |s| <= 200000`
- `s` contains printable ASCII characters

## Clarifying Notes

- Leading, trailing, and multiple spaces are treated as separators and are not preserved.

## Example Input

```
the quick brown fox
```

## Example Output

```
fox brown quick the
```
