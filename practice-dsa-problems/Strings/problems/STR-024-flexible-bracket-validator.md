# Flexible Bracket Validator

## Problem Statement

Given a string containing `(`, `)`, and `*`, determine if it can be interpreted as a valid parentheses sequence. Each `*` may be treated as `(`, `)`, or an empty string.

## Input Format

- Single line: string `s`

## Output Format

- `true` or `false`

## Constraints

- `1 <= |s| <= 200000`

## Clarifying Notes

- A valid parentheses sequence has balanced and properly nested parentheses.

## Example Input

```
(*)
```

## Example Output

```
true
```
