# STR-006: Single-Edit Symmetry

## Problem Statement

Given a string `s`, determine whether it can become a palindrome by removing at most one character. Output `true` if possible, otherwise `false`.

## Input Format

- Single line: string `s`

## Output Format

- `true` or `false`

## Constraints

- `1 <= |s| <= 200000`
- `s` contains printable ASCII characters

## Clarifying Notes

- Removing zero characters is allowed.
- The check is case-sensitive.

## Example Input

```
abcda
```

## Example Output

```
true
```
