# STR-010: Boundary Match Counter

## Problem Statement

Given a string `s`, count the number of substrings whose first and last characters are the same.

## Input Format

- Single line: string `s`

## Output Format

- Single integer: the count

## Constraints

- `1 <= |s| <= 200000`
- `s` contains printable ASCII characters

## Clarifying Notes

- Substrings are contiguous and non-empty.
- The answer can be large; use 64-bit integers.

## Example Input

```
abcab
```

## Example Output

```
7
```
