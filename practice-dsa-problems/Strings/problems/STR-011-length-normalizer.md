# Length Normalizer

## Problem Statement

Given a string `s`, you may insert characters anywhere. Find the minimum number of insertions needed to make the string length even.

## Input Format

- Single line: string `s`

## Output Format

- Single integer: minimum insertions needed

## Constraints

- `1 <= |s| <= 200000`
- `s` contains printable ASCII characters

## Clarifying Notes

- You do not need to output the modified string.

## Example Input

```
abc
```

## Example Output

```
1
```
