# STR-008: Uniform Frequency Text

## Problem Statement

Given a string `s`, determine whether all distinct characters appear the same number of times. Output `true` if they do, otherwise `false`.

## Input Format

- Single line: string `s`

## Output Format

- `true` or `false`

## Constraints

- `1 <= |s| <= 200000`
- `s` contains printable ASCII characters

## Clarifying Notes

- Character comparison is case-sensitive.

## Example Input

```
aabbcc
```

## Example Output

```
true
```
