# StringsClassic Problems

This folder contains classic string algorithm problems following the Universal DSA Problem Generation structure.

## Folder Structure

```
StringsClassic/
├── problems/           # Problem statements (STC-XXX-[slug].md)
├── editorials/         # Complete editorials with solutions (STC-XXX-[slug].md)
├── testcases/          # Test cases in YAML format (STC-XXX-[slug].yaml)
├── quizzes/            # Quiz questions (STC-XXX-[slug].yaml)
└── images/             # Image generation prompts (STC-XXX/README.md)
```

## Naming Convention

- **Prefix:** STC
- **Format:** STC-XXX-[kebab-case-slug]
- **Problem ID:** STC_[DESCRIPTION_CAPS]__[4_DIGITS]
