# StringsClassic Images Directory

Place image generation prompt directories here.

**Format:** `STC-XXX/README.md`

**Example:** `STC-001/README.md`

## Directory Structure

Each problem should have its own subdirectory:

```
images/
├── STC-001/
│   └── README.md
├── STC-002/
│   └── README.md
└── STC-003/
    └── README.md
```

## Required Images (per problem)

1. **problem-illustration.png** - Main visual of problem concept
2. **real-world-scenario.png** - Real-world application visualization
3. **algorithm-visualization.png** - Algorithm flow diagram
4. **algorithm-steps.png** - Step-by-step execution
5. **example-1.png** - Example walkthrough

## Image Specifications

- Format: PNG
- Resolution: 1200x800px (standard), 1600x900px (complex)
- Style: Technical documentation
- Colors: Blues (#2563eb), Greens (#059669), Grays (#6b7280)

## README Template

Each `STC-XXX/README.md` should contain generation prompts for all required images.

See `UNIVERSAL_DSA_GENERATION_PROMPT.md` for complete image generation template.
