# Greedy Problems

This folder contains greedy algorithm problems following the Universal DSA Problem Generation structure.

## Folder Structure

```
Greedy/
├── problems/           # Problem statements (GRD-XXX-[slug].md)
├── editorials/         # Complete editorials with solutions (GRD-XXX-[slug].md)
├── testcases/          # Test cases in YAML format (GRD-XXX-[slug].yaml)
├── quizzes/            # Quiz questions (GRD-XXX-[slug].yaml)
└── images/             # Image generation prompts (GRD-XXX/README.md)
```

## Naming Convention

- **Prefix:** GRD
- **Format:** GRD-XXX-[kebab-case-slug]
- **Problem ID:** GRD_[DESCRIPTION_CAPS]__[4_DIGITS]

## Example Files

- `problems/GRD-001-activity-selection.md`
- `editorials/GRD-001-activity-selection.md`
- `testcases/GRD-001-activity-selection.yaml`
- `quizzes/GRD-001-activity-selection.yaml`
- `images/GRD-001/README.md`
