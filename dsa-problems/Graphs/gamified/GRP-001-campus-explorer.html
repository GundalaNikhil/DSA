Objective: Explore all buildings starting from the Main Entrance using BFS.

How BFS Works:

  • Start at node 0 (Main Entrance)
  • Add it to the queue
  • Visit the front of queue, add unvisited neighbors
  • Repeat until queue is empty

Key Insight: BFS visits nodes level-by-level (closest first)!
