# ASCII Visual References for GEO-008: Minimum Enclosing Circle

## Overview

Plain ASCII sketches to illustrate enclosing circles.

## ASCII Snippets

### Right triangle example
```
● (0,0)
|\
| \
|  \
●   ● (1,0)
(0,1)

Circle centered at (0.5,0.5) radius ≈ 0.707107 covers all points.
```

### Two-point case
```
●---------●
(-3,1)   (3,1)
Center (0,1), radius 3
```

### Rectangle corners
```
● (0,0) ------- ● (4,0)
|                  |
|                  |
● (0,4) ------- ● (4,4)
Center (2,2), radius ≈ 2.828427 (half diagonal)
```
