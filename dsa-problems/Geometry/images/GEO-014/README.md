# ASCII Visual References for GEO-014: Angle Sorting for Polar Order

## Overview

Plain ASCII sketches to show angular ordering around the origin.

## ASCII Snippets

### Basic quadrants
```
      ● (0,1)
       \
        \
         O ---- ● (1,0)
        /
       /
      ● (-1,0)

Order (CCW): (1,0) → (0,1) → (-1,0) → ... lower half last.
```

### Tie on angle, distance tie-break
```
● (1,1)   ● (2,2)  (same angle 45°)

Closer one (1,1) goes before (2,2).
```
