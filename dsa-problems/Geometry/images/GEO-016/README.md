# ASCII Visual References for GEO-016: Manhattan MST

## Overview

Plain ASCII sketches to illustrate Manhattan MST edges.

## ASCII Snippets

### Simple triangle
```
● (0,0)   ● (3,0)
     ● (2,2)

Distances:
 (0,0)-(3,0) = 3
 (0,0)-(2,2) = 4
 (3,0)-(2,2) = 3
MST = 3 + 3 = 6
```

### Grid points
```
● (0,0)   ● (2,0)
● (0,2)   ● (2,2)

Best edges connect neighbors; total = 6.
```
