## Problem Description

Find maximum matching in a tree. A matching is a set of edges with no shared vertices.

## Input Format

- Line 1: N
- Next N-1 lines: u v (edges)

## Output Format

Maximum matching size.

## Examples

### Example 1

**Input:**

```
4
1 2
2 3
3 4
```

**Output:**

```
2
```

### Example 2

**Input:**

```
7
1 2
1 3
2 4
2 5
3 6
3 7
```

**Output:**

```
3
```

## Constraints

- 1 ≤ N ≤ 200,000
