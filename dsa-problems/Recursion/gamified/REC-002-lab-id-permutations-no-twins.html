The Problem:

Generate all unique permutations where no two identical characters are adjacent.

Backtracking Strategy:

1. Sort the string first
2. At each position, try each unused character
3. Skip if same as previous character (to avoid duplicates)
4. Prune: Don't place if it creates adjacent twins!
