üí° Key Concepts

Two Pointers

Start from both ends (index 1 and n-2) and move inward until they meet

In-Place Swap

Swap elements directly in the array without using extra space

Time: O(n)

Visit each element once, requiring roughly n/2 swaps

Space: O(1)

Only need two pointer variables, no matter how big the array is
