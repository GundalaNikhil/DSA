# Stacks Problems

This folder contains stack-related problems following the Universal DSA Problem Generation structure.

## Folder Structure

```
Stacks/
├── problems/           # Problem statements (STK-XXX-[slug].md)
├── editorials/         # Complete editorials with solutions (STK-XXX-[slug].md)
├── testcases/          # Test cases in YAML format (STK-XXX-[slug].yaml)
├── quizzes/            # Quiz questions (STK-XXX-[slug].yaml)
└── images/             # Image generation prompts (STK-XXX/README.md)
```

## Naming Convention

- **Prefix:** STK
- **Format:** STK-XXX-[kebab-case-slug]
- **Problem ID:** STK_[DESCRIPTION_CAPS]__[4_DIGITS]

## Example Files

- `problems/STK-001-valid-parentheses.md`
- `editorials/STK-001-valid-parentheses.md`
- `testcases/STK-001-valid-parentheses.yaml`
- `quizzes/STK-001-valid-parentheses.yaml`
- `images/STK-001/README.md`
