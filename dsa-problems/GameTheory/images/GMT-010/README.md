# Image Generation Prompts for GMT-010: Removal Game on Strings

## Required Images

### 1. problem-illustration.png

**Purpose:** Main visual of problem concept

**Generation Prompt:**
```
Create a technical diagram showing String Compression and Removal:
- String: "AAABBAA" (colored blocks).
- Step 1: Compress to "A - B - A".
- Step 2: Remove "B".
- Step 3: "A" and "A" merge into single "A".
- Clean, professional style.
- Size: 1200x800px
```

### 2. real-world-scenario.png

**Purpose:** Real-world application visualization

**Generation Prompt:**
```
Illustrate the Tunnel Collapse scenario:
- A tunnel made of segments (Concrete, Steel, Concrete).
- Removing the Steel segment causes the two Concrete segments to crash together and fuse.
- Dramatic but technical style.
- Size: 1200x800px

**Purpose:** Algorithm flow diagram

```
