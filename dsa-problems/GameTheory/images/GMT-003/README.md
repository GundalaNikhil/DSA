# Image Generation Prompts for GMT-003: Subtract-a-Square with Ban List

## Required Images

### 1. problem-illustration.png

**Purpose:** Main visual of problem concept

**Generation Prompt:**
```
Create a technical diagram showing the Subtract-a-Square Game:
- Display a pile of 7 stones.
- Show possible moves: "-4" (Valid, Square), "-1" (Invalid, Banned).
- Icon for "Banned Set" containing the number 1.
- Clean, professional style with blue/gray tones.
- Size: 1200x800px
```

### 2. real-world-scenario.png

**Purpose:** Real-world application visualization

**Generation Prompt:**
```
Illustrate the Restricted Budget scenario:
- Context: A financial dashboard showing a deficit.
- Action: Cutting budget in fixed blocks (squares).
- Constraint: Some blocks (like $1k) are crossed out/locked.
- Metaphor: "Strategic cuts to reach zero".
- Modern professional style.
- Size: 1200x800px
```

### 3. algorithm-visualization.png

**Purpose:** Algorithm flow diagram

**Generation Prompt:**
```
Create flowchart showing DP State Transitions:
- Array indices 0 to 7.
- Arrows from i to i-s (e.g., 7 -> 3).
- Color coding: Green for Winning (True), Red for Losing (False).
- Logic: "If any arrow points to Red, current is Green".
- Size: 1600x900px
```

### 4. example-1.png

**Purpose:** Example walkthrough

**Generation Prompt:**
```
Visualize the example n=7, B={1}:
- Start at 7.
- Only valid move: -4.
- Transition to 3.

- Outcome: 3 is Losing -> 7 is Winning.
- Size: 1200x800px
```
