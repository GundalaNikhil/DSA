# Trees Problems

This folder contains tree data structure problems following the Universal DSA Problem Generation structure.

## Folder Structure

```
Trees/
├── problems/           # Problem statements (TRE-XXX-[slug].md)
├── editorials/         # Complete editorials with solutions (TRE-XXX-[slug].md)
├── testcases/          # Test cases in YAML format (TRE-XXX-[slug].yaml)
├── quizzes/            # Quiz questions (TRE-XXX-[slug].yaml)
└── images/             # Image generation prompts (TRE-XXX/README.md)
```

## Naming Convention

- **Prefix:** TRE
- **Format:** TRE-XXX-[kebab-case-slug]
- **Problem ID:** TRE_[DESCRIPTION_CAPS]__[4_DIGITS]

## Example Files

- `problems/TRE-001-binary-tree-inorder.md`
- `editorials/TRE-001-binary-tree-inorder.md`
- `testcases/TRE-001-binary-tree-inorder.yaml`
- `quizzes/TRE-001-binary-tree-inorder.yaml`
- `images/TRE-001/README.md`

## Common Tree Topics

- Tree Traversals (Inorder, Preorder, Postorder, Level Order)
- Binary Search Tree Operations
- Lowest Common Ancestor
- Tree Diameter
- Tree Height/Depth
- Path Sum Problems
- Symmetric/Mirror Trees
- Vertical/Top/Bottom View
- Serialize/Deserialize
- Balance Checking
